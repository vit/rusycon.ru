Object-Oriented Modeling Complex for Pump Unit Control System

The pump unit control system (PUCS) is developed in the Educational Center of Automation of Robotics and Manufacturing System Automation Department. This system allows to control pump units at the up-to-date level. PUCS has user interface and provides possibility of extension of its functionality. The object-oriented model of PUCS consists of two programs:

The program for visualisation of the algorithm of the pump unit start-up exhibits main stages of the start-up and shut-down modes. Working blocks are marked in red. Working sections of pipelines are marked in blue.

The program for calculation of electrical energy saving shows the bar-diagrams of difference between energy consumption achieved by in-valve throttling control (red) and energy consumption achieved by frequency-controlled AC drive (blue). The values of water flow rate Q (in the range from 0 to 320 m3/hour) in 6 time intervals can be changed by user.
