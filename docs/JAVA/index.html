Объектно-ориентированный моделирующий комплекс системы управления насосными агрегатами

В учебном центре компьютерных средств автоматизации кафедры РАПС С.-Петербургского гос. электротехнического университета разработана система управления, позволяющая на качественно новом уровне управлять технологическим оборудованием насосного агрегата. Подсистема поддерживает ЧМИ и обеспечивает возможность наращивания функциональных задач. Объектно-ориентированный пример системы управления насосными агрегатами состоит из двух частей:

Программа визуализации алгоритма запуска насосного агрегата показывает последовательность процессов, происходящих при запуске и останове насосного агрегата. Устройства обозначаются красным цветом, если они работают, либо черным, если они выключены. В промежуточных состояниях происходит переключение красного и черного цветов. Рабочие участки обозначены синим цветом, нерабочие - черным. Внизу экрана располагается подсказка, в которой описывается происходящий в данный момент процесс.

Программа расчета экономии электроэнергии основана на определении разницы между величинами потребления электроэнергии при регулировании напора насоса путем дросселирования напорной задвижкой и при регулировании с помощью частотно-регулируемого электропривода переменного тока на примере насоса Д 320 - 70а. В программу заложена возможность ввода пользователем значения расхода воды Q (от 0 до 320) в различные временные интервалы, на основании чего программа выводит бар-диаграмму потребляемых при различных методах мощностей и зависимость сэкономленной электроэнергии от времени. На бар-диаграмме красным цветом показана мощность, потребляемая при регулировании напора насоса путем дросселирования напорной задвижкой, синим - при регулировании с помощью частотно-регулируемого электропривода переменного тока.
