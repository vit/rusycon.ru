М.С. Финкельштейн. Модели случайных точечных процессов в задачах анализа надежности и безопасности (176 с.)
Издательство ЦНИИ "Электроприбор"
Рассматриваются вопросы применения простейших точечных процессов для решения задач надежности и безопасности. Базовыми для исследования различных приложений являются неоднородный пуассоновский процесс и процесс восстановления.

Разработанные методы применяются для оценки надежности математического обеспечения и безопасности прохождения объектом заданного маршрута. Впервые приводится математическое изложение теории неполного восстановления, основанной на введенном автором обобщенном процессе восстановления. Подробно рассмотрены различные приложения этой теории для решения задач, возникающих при планировании ускоренных испытаний и при оценке влияния внешних неблагоприятных воздействий на функционирующую систему. Книга рассчитана на специалистов, занимающихся вопросами оценки и обеспечения качества функционирования различных объектов.

Библиогр.: 146 назв. Ил. 10. Табл. 2.
