А.Е. Барабанов. Синтез минимаксных регуляторов.
СПб: Издательство СПбГУ, 1996
В монографии разработана теория оптимальной стабилизации линейного объекта управления с постоянно действующими детерминированными возмущениями из заданного класса последовательностей как минимаксный аналог теории Винера - Колмогорова. Дается определение и исследуются свойства стационарного множества последовательностей, заменяющего случайный стационарный процесс в стохастической теории управления. Решается минимаксная задача, являющаяся одновременно игрой двух лиц с равномерным по времени функционалом качества на неограниченном интервале.

Книга предназначена для научных и инженерно-технических работников, специализирующихся в области управления.
